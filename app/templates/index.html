Flow Manager

Flow Manager es un prototipo de interface de creacion de flujos desarroda para el controlador SDN OpenDayLight versión Carbon. Mediante él se podrá crear flujos de para el manejo de la red gestionada por el controlador. Las pruebas realizadas se efecturan sobre una red virtualizada mediante la herramienta Mininet. Al momento de la insercion de flujos se debera ingresar al menos una de las siguientes opciones: IP origen, IP destino, Puerto Origen y Puerto Destino.
